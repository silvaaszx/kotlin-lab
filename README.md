## 🔬 Sobre o Laboratório

Repositório dedicado ao aprimoramento contínuo no ecossistema Android. Aqui guardo os meus fragmentos de código, desafios lógicos em Kotlin, testes de injeção de dependências e implementações isoladas de UI.

## 📁 Estrutura

Cada pasta contém um mini-projeto ou teste técnico:
* Nomenclatura das pastas: `[Nome-do-Estudo]` (Ex: `/Compose-Animations`, `/Room-Database-Setup`, `/Coroutines-Playground`)
* Abra a pasta desejada diretamente no **Android Studio** para que o Gradle sincronize as dependências corretamente.
